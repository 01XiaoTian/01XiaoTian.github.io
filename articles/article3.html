PDF密码破解 - 一键解除限制

发布于 2024年10月5日 作者：阿坑

之前阿坑写过一篇文章PDF解密神器，一键解除限制！，讲述了如何去除PDF密码，不过只能去除限制编辑、复制等功能的密码。那么，如果连PDF都不开，那该怎么办？思路和这篇去除压缩包密码的思路一样。

01、如何加密？

解密之前咱们先说说如何加密，以Adobe Acrobat Pro DC这款软件为例。点击菜单栏的编辑，选择保护、安全性属性。在安全性方法中，选择口令安全性。

口令安全性包括文档打开口令和文档许可口令，其中文档打开口令要求打开文档的口令，而文档许可口令限制文档的编辑和打印等权限。选择文档打开口令，并设置密码。

02、如何解密？

与上一篇文章推荐的软件相似，阿坑找到了这款名为Passper for PDF的软件，获取方式在文末。这款软件可以帮助大家快速地恢复PDF文件的打开密码以及编辑权限密码。具有非常人性化的引导操作，轻松上手。

首先，将压缩包解压缩后，点击【Setup.exe】进行安装，安装成功关闭软件。将【Crack UZ1】文件夹下的补丁【Passper for PDF】复制到上一步操作的安装路径中。

此安装路径可通过，单击桌面上的软件图标，再单击【属性】获取。之后，再次点击桌面上的软件图标即可打开软件，注意：不要更新，直接关闭这个弹窗即可。

上传PDF文件后，软件会自动分析PDF文件的加密情况。PDF密码恢复的方法有组合、字典、掩码、暴力四种方法，与之前提到压缩包的一样，再见了，压缩包密码！，这里就不再解释了。选择相应的方法后，即可运行，每秒尝试的密码数高达5000多个。5分钟，轻松获取6位数的纯数字密码。

03、代码实现

除了使用现成的软件外，还可以自己编写代码，通过遍历密码字典，进行尝试获取。使用Python代码生成了包含000000-999999的六位数纯数字密码的txt文档，其他位数同理。

# 创建一个Python脚本，用于生成从000000到999999的六位数
# 并保存到txt文档中
with open('/mnt/data/six_digit_numbers.txt', 'w') as file:
    for number in range(1000000):
        formatted_number = f"{number:06d}"
        file.write(formatted_number + '\n')

使用Python代码生成了包含英文字母的四位纯字母密码的txt文档，其他位数同理。

import string

# 创建一个Python脚本，用于生成从aaaa到zzzz的所有四位英文字母组合，并保存到txt文档中
with open('four_letter_combinations.txt', 'w') as file:
    for first_letter in string.ascii_lowercase:
        for second_letter in string.ascii_lowercase:
            for third_letter in string.ascii_lowercase:
                for fourth_letter in string.ascii_lowercase:
                    combination = first_letter + second_letter + third_letter + fourth_letter
                    file.write(combination + '\n')

于是，我们得到了生成的字典txt文件，编写剩余代码，同时使用设置了113579为密码的测试PDF。

# 也许还需要安装pip install pycryptodome
import PyPDF2

def read_passwords(password_file, encoding='latin1'):
    with open(password_file, 'rb') as file:
        passwords = file.readlines()
    return [password.decode(encoding).strip() for password in passwords]

def decrypt_pdf(pdf_file, password_file):
    passwords = read_passwords(password_file)
    with open(pdf_file, 'rb') as file:
        reader = PyPDF2.PdfReader(file)
        if not reader.is_encrypted:
            print("The PDF file is not encrypted.")
            return
        for password in passwords:
            try:
                reader.decrypt(password)
                reader.pages[0]
                print(f"Password found: {password}")
                return
            except Exception as e:
                print(f"Tried password: {password} - Failed ({str(e)})")
    print("No valid password found in the dictionary.")

if __name__ == "__main__":
    pdf_file = "TestPdf113579.pdf"  # PDF文件路径
    password_file = "six_digit_numbers.txt"  # 密码字典文件路径
    decrypt_pdf(pdf_file, password_file)
